## Definition of Done

A repository is in strong condition when a new human contributor or coding agent can, using only version-controlled information:

- [ ] understand the project's purpose and architecture;
- [ ] install the correct runtime and dependencies;
- [ ] configure the local environment without receiving secrets through source control;
- [ ] start the project through a documented command;
- [ ] identify where a change belongs and which conventions apply;
- [ ] format, lint, type-check, test, and build the change;
- [ ] run integration or end-to-end validation when needed;
- [ ] understand ownership, contribution, and security-reporting rules;
- [ ] open a pull request that is validated by CI;
- [ ] rely on protected branches and automated maintenance safeguards.
